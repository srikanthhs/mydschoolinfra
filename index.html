Fix Firebase Access
Follow these steps in Firebase Console
1Open Firebase Console
Go to console.firebase.google.com
Select project mydschoolinfra
2Create Firestore Database
Left menu → Build → Firestore Database
If not created: click Create database → choose Start in test mode → select asia-south1 (Mumbai)
3Set Rules
Inside Firestore Database → click Rules tab
Replace all text with:
service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
Click Publish
✅ After publishing rules
The app will automatically reconnect and load data within a few seconds.
